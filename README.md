<h1>BelanjaIn Backend API</h1> <p>Backend untuk aplikasi e-commerce BelanjaIn. Dibangun dengan Node.js, Express.js, dan PostgreSQL.</p> <hr> <h2>Cara Menjalankan</h2> <p><b>1. Clone repository</b></p> <pre> git clone https://github.com/galactossa/belanjain-backend.git cd belanjain-backend </pre> <p><b>2. Install dependencies</b></p> <pre> npm install </pre> <p><b>3. Setup environment variable</b></p> <p>Buat file <code>.env</code> di root folder:</p> <pre> DB_USER=postgres DB_PASSWORD=password_mu DB_HOST=localhost DB_PORT=5432 DB_NAME=BelanjaIn PORT=3000 JWT_SECRET=rahasia_jwt_kamu </pre> <p><b>4. Jalankan server</b></p> <pre> npm run dev </pre> <p>Server berjalan di: <b>http://localhost:3000</b></p> <hr> <h2>Daftar Endpoint API</h2> <h3>Autentikasi (Pembeli)</h3> <ul> <li>POST /api/pengguna/register - Registrasi akun baru</li> <li>POST /api/pengguna/login - Login ke akun</li> </ul> <p><b>Contoh Request Register:</b></p> <pre> { "nama": "Tora Eldoardo Eko Putra", "email": "tora@gmail.com", "password": "123456", "telepon": "08123456789" } </pre> <p><b>Contoh Response Login:</b></p> <pre> { "message": "Login berhasil", "token": "eyJhbGciOiJIUzI1NiIs...", "pengguna": { "id_pengguna": 1, "nama": "Tora Eldoardo Eko Putra", "email": "tora@gmail.com", "role": "pembeli" } } </pre> <hr> <h3>Profil dan Alamat (Pembeli)</h3> <ul> <li>PUT /api/pengguna/:id - Edit profil</li> <li>POST /api/alamat - Tambah alamat</li> <li>PUT /api/alamat/:id - Edit alamat</li> <li>DELETE /api/alamat/:id - Hapus alamat</li> <li>GET /api/alamat/pengguna/:id_pengguna - Lihat semua alamat</li> </ul> <hr> <h3>Produk</h3> <ul> <li>GET /api/produk - Lihat semua produk (pagination)</li> <li>GET /api/produk/:id - Detail produk</li> <li>GET /api/produk/search?q=nama - Cari produk</li> <li>GET /api/produk/filter?min_harga=&max_harga=&sort= - Filter produk</li> <li>GET /api/produk/suggestions?q=keyword - Auto-suggest</li> <li>POST /api/produk - Tambah produk (penjual)</li> <li>PUT /api/produk/:id - Edit produk (penjual)</li> <li>DELETE /api/produk/:id - Hapus produk (penjual)</li> </ul> <p><b>Parameter Filter:</b></p> <ul> <li>min_harga</li> <li>max_harga</li> <li>id_kategori</li> <li>sort: harga_terendah, harga_tertinggi, terlaris, rating_tertinggi</li> </ul> <hr> <h3>Wishlist</h3> <ul> <li>POST /api/wishlist - Tambah</li> <li>DELETE /api/wishlist/:id_pengguna/:id_produk - Hapus</li> <li>GET /api/wishlist/pengguna/:id_pengguna - Lihat</li> </ul> <hr> <h3>Keranjang</h3> <ul> <li>POST /api/keranjang - Tambah</li> <li>PUT /api/keranjang/:id - Update jumlah</li> <li>DELETE /api/keranjang/:id - Hapus item</li> <li>DELETE /api/keranjang/pengguna/:id_pengguna/clear - Kosongkan keranjang</li> <li>GET /api/keranjang/pengguna/:id_pengguna - Lihat keranjang</li> </ul> <hr> <h3>Pesanan dan Checkout</h3> <ul> <li>POST /api/pesanan - Checkout</li> <li>GET /api/pesanan/pengguna/:id_pengguna - Riwayat</li> <li>GET /api/pesanan/:id - Detail</li> <li>PUT /api/pesanan/:id/status - Update status</li> <li>PUT /api/pesanan/:id/pembayaran - Konfirmasi pembayaran</li> </ul> <p><b>Status:</b> menunggu → diproses → dikirim → dalam_perjalanan → selesai</p> <hr> <h3>Voucher</h3> <ul> <li>GET /api/voucher - Lihat semua</li> <li>GET /api/voucher/kode/:kode - Cek voucher</li> <li>POST /api/voucher - Tambah (admin)</li> <li>PUT /api/voucher/:id - Edit (admin)</li> <li>DELETE /api/voucher/:id - Hapus (admin)</li> </ul> <hr> <h3>Review dan Rating</h3> <ul> <li>POST /api/ulasan - Tambah review</li> <li>GET /api/ulasan/produk/:id_produk - Lihat review</li> <li>DELETE /api/ulasan/:id - Hapus review</li> </ul> <hr> <h3>Trust Score</h3> <ul> <li>GET /api/trust-score/produk/:id_produk - Lihat trust score</li> <li>GET /api/trust-score/semua - Semua trust score</li> </ul> <hr> <h3>Notifikasi</h3> <ul> <li>GET /api/notifikasi/pengguna/:id_pengguna - Lihat notifikasi</li> <li>PUT /api/notifikasi/:id/read - Tandai dibaca</li> <li>POST /api/notifikasi/broadcast - Broadcast (admin)</li> </ul> <hr> <h3>Statistik dan Dashboard</h3> <ul> <li>GET /api/statistik/admin - Dashboard admin</li> <li>GET /api/statistik/penjual/:id_toko - Dashboard penjual</li> </ul> <hr> <h3>Manajemen Toko (Penjual)</h3> <ul> <li>POST /api/toko - Upgrade jadi penjual</li> <li>PUT /api/toko/:id - Edit profil toko</li> <li>GET /api/toko/:id/produk - Lihat produk toko</li> </ul> <hr> <h3>Laporan (Report System)</h3> <ul> <li>POST /api/laporan - Buat laporan</li> <li>GET /api/laporan?role=admin - Lihat semua (admin)</li> <li>PUT /api/laporan/:id/status - Validasi laporan</li> </ul> <hr> <h3>Kategori (Admin)</h3> <ul> <li>GET /api/kategori - Lihat semua</li> <li>POST /api/kategori?role=admin - Tambah</li> <li>PUT /api/kategori/:id?role=admin - Edit</li> <li>DELETE /api/kategori/:id?role=admin - Hapus</li> </ul> <hr> <h3>Transaksi (Admin)</h3> <ul> <li>GET /api/transaksi - Lihat semua transaksi</li> <li>GET /api/transaksi/:id - Detail transaksi</li> </ul> <hr> <h3>Multi-Admin (Super Admin)</h3> <ul> <li>GET /api/admin - Lihat semua admin</li> <li>POST /api/admin - Tambah admin</li> <li>PUT /api/admin/:id - Edit admin</li> <li>DELETE /api/admin/:id - Hapus admin</li> </ul> <hr> <h3>System Settings (Super Admin)</h3> <ul> <li>GET /api/settings - Lihat semua</li> <li>GET /api/settings/:key - Lihat by key</li> <li>POST /api/settings - Tambah</li> <li>PUT /api/settings/:key - Update</li> <li>DELETE /api/settings/:key - Hapus</li> </ul> <p><b>Default Settings:</b></p> <ul> <li>privacy_policy</li> <li>about_us</li> <li>contact_info</li> <li>office_address</li> <li>company_logo</li> </ul> <hr> <h3>Blog (Admin)</h3> <ul> <li>GET /api/blogs - Lihat semua artikel</li> <li>GET /api/blogs/:id - Detail artikel</li> <li>POST /api/blogs - Tambah artikel</li> <li>PUT /api/blogs/:id - Edit artikel</li> <li>DELETE /api/blogs/:id - Hapus artikel</li> </ul> <hr> <h3>Search History (Pembeli)</h3> <ul> <li>GET /api/search-history/pengguna/:id_pengguna - Lihat riwayat</li> <li>POST /api/search-history - Simpan keyword</li> <li>DELETE /api/search-history/:id - Hapus satu</li> <li>DELETE /api/search-history/pengguna/:id_pengguna/clear - Hapus semua</li> </ul> <hr> <h3>Loyalty Points & Membership</h3> <ul> <li>GET /api/loyalty/points/pengguna/:id_pengguna - Total poin</li> <li>GET /api/loyalty/history/pengguna/:id_pengguna - Riwayat poin</li> <li>GET /api/loyalty/membership/pengguna/:id_pengguna - Level membership</li> <li>POST /api/loyalty/points - Tambah poin</li> <li>PUT /api/loyalty/points/redeem - Tukar poin</li> </ul> <p><b>Level Membership:</b></p> <ul> <li>Regular (0–499) 🟢</li> <li>Gold (500–999) 🥇</li> <li>Platinum (1000+) 💎</li> </ul> <hr> <h2>Tech Stack</h2> <ul> <li>Node.js</li> <li>Express.js</li> <li>PostgreSQL</li> <li>pg</li> <li>dotenv</li> <li>cors</li> <li>bcrypt</li> <li>jsonwebtoken (JWT)</li> </ul> <hr> <h2>Role dan Akses</h2> <ul> <li><b>Admin:</b> Semua endpoint</li> <li><b>Penjual:</b> Kelola produk & pesanan</li> <li><b>Pembeli:</b> Belanja & fitur user</li> </ul> <hr> <h2>Catatan</h2> <ul> <li>Gunakan header: Authorization: Bearer &lt;token&gt;</li> <li>Password di-hash dengan bcrypt</li> <li>JWT berlaku 7 hari</li> </ul> <hr>
